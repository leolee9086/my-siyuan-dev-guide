## CSS 代码片段示例

下面是一个简单的 CSS 片段示例，演示了如何通过 `code-group` 和 `snippet-meta` 来定义和管理片段：

::: code-group
```css [让所有的div变成红色]
div{
    color:red
}
```
```json [snippet-meta]
{
    "type":"css",
    "id":"from-leolee9086-test1",
    "name":"全部div变红"
}
```
:::
