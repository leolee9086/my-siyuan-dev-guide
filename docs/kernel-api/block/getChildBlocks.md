# 获取子块

需要认证

## 接口描述

获取指定块的所有直接子块，不包括孙子块。

## 请求参数

| 参数名 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| id | string | 是 | 父块ID |

## 返回值

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 状态码，0表示成功 |
| msg | string | 消息，用于显示错误信息 |
| data | array | 子块数组 |
