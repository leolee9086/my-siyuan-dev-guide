# 折叠块

需要认证

## 接口描述

折叠指定的块，隐藏其子块内容。

## 请求参数

| 参数名 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| id | string | 是 | 要折叠的块ID |

## 返回值

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 状态码，0表示成功 |
| msg | string | 消息，用于显示错误信息 |
| data | null | 返回数据，成功操作时为null |
