/api/asset/statAsset

# 获取资源统计信息

需要认证 需要管理员权限

## 接口描述

获取指定资源文件的统计信息，如大小、修改时间等。

## 请求参数

（待补充详细参数）

| 参数名 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| path | string | 是 | 资源文件的路径 (相对于 assets 目录) |

## 返回值

返回资源的统计信息。

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| code | number | 返回码，0表示成功 |
| msg | string | 返回信息 |
| data | object | 包含 \`size\` (大小，字节) 和 \`modTime\` (修改时间戳，毫秒) 等字段的统计信息对象 |

请求示例

返回示例

### 请求示例

```
{
    "path": "assets/image-20231027132000-xyz.png"
}
```

### 返回示例

```
{
    "code": 0,
    "msg": "",
    "data": {
        "size": 12345,
        "modTime": 1698384000000 
    }
}
```
